# STIG Checker

Paste your AndroidManifest.xml and get informed of STIG issues.

## Usage
1. Start the app with `npm start`.
2. Paste your AndroidManifest.xml.
3. Click "Check STIG" to see issues.

## Features
- Manifest input
- STIG checks (expandable)
- Issue reporting
